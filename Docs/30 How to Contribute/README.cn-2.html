• 你也可以通过以下方式在本地测试文档：

    1. 更改文档。

    2. 在存储库的根目录下，运行：dotnet fsi build.fsx -- -p docs。

    3. 运行：dotnet run --project .\Fun.Blazor.Docs.Server\Fun.Blazor.Docs.Server.fsproj。
